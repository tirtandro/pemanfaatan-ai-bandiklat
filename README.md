# Materi Fisika Interaktif: Teori Relativitas Khusus Einstein

Proyek ini adalah sebuah aplikasi web edukasi interaktif yang membahas tentang **Teori Relativitas Khusus Einstein**. Dirancang dengan estetika modern dan fitur interaktif untuk memudahkan pemahaman konsep fisika yang kompleks bagi siswa SMA Kelas XII.

## 🚀 Fitur Utama

- **Layar Pemuatan (Loading Screen) Animasi**: Animasi atom dan partikel yang menarik saat aplikasi pertama kali dimuat.
- **Efek Parallax & Latar Belakang Luar Angkasa**: Desain visual premium dengan latar belakang bintang berkelip yang memberikan kesan kedalaman.
- **Animasi SVG Terintegrasi**: 
    - Visualisasi gelombang cahaya.
    - Jam cahaya untuk dilatasi waktu.
    - Kontraksi panjang pada pesawat roket.
    - Kesetaraan massa-energi ($E = mc^2$).
    - Orbit satelit GPS dan peluruhan muon.
- **Kalkulator Relativistik Real-time**: Memungkinkan pengguna menghitung faktor Lorentz, dilatasi waktu, kontraksi panjang, massa relativistik, dan energi total secara instan.
- **Grafik Interaktif**: Visualisasi grafik Faktor Lorentz ($\gamma$) terhadap kecepatan ($v/c$).
- **Kuis Interaktif**: Uji kompetensi dengan 15 soal acak lengkap dengan pembahasan.
- **Desain Responsif**: Optimal untuk tampilan mobile maupun desktop dengan gaya *Glassmorphism*.

## 🛠️ Teknologi yang Digunakan

- **HTML5 & CSS3** (Vanilla CSS dengan Tailwind CSS via CDN)
- **JavaScript** (Vanilla JS)
- **MathJax**: Untuk peragaan rumus matematika yang presisi.
- **Chart.js**: Untuk visualisasi grafik.
- **Font Awesome**: Untuk ikon-ikon menarik.
